# City Explorer Learning Project

Bu repo, modern web gelistirme konularini adim adim ogrenmek icin birlikte ilerlettigimiz egitim projesidir.

## Proje Amaci

Bu projede modern bir web uygulamasi gelistirecegiz. Uygulamanin ana fikri:

- Kullanici bir sehir sececek
- Uygulama o sehirle ilgili bilgiler gosterecek
- Daha sonra hava durumu ve diger dinamik veriler API uzerinden cekilecek
- Surec boyunca JavaScript, HTML, CSS, DOM, eventler ve API kullanimi ogrenilecek

## Egitim Kurallari

Proje su prensiplerle ilerleyecek:

- Her konu kucuk ve bagimsiz parcalara ayrilacak
- Once konsept anlatilacak
- Sonra ornek kod gosterilecek
- Ardindan kisa uygulamali alistirma verilecek
- Her parcayi bitirdikten sonra bir sonraki adima gecilecek

## Konu Sirasi

Projede su sira izlenecek:

1. JavaScript ve temel yapilar
2. Ileri JavaScript
3. HTML ve CSS
4. DOM ve eventler
5. Axios ve API ile veri cekme

## Proje Konsepti

Projenin adi: `City Explorer`

Ilk surumde temel sayfa iskeleti kuruldu. Sonraki adimlarda uygulama asama asama gelisecek:

- sehir listesi
- secilen sehre gore icerik gosterimi
- DOM manipülasyonu
- kullanici etkilesimi
- API'den veri cekme

## Mevcut Durum

Su anda tamamlanan kisim:

- Projenin temel HTML iskeleti kuruldu
- CSS ile modern bir acilis ekrani hazirlandi
- `script.js` dosyasi projeye baglandi
- Proje konsepti netlestirildi

## Bulundugumuz Adim

Mevcut adim: `Adim 1 - Konsept tasarimi ve temel iskelet`

Bu adimda ogrenilenler:

- HTML, CSS ve JavaScript dosyalarinin rolu
- Basit ama duzenli bir proje yapisi kurma
- Sonraki adimlar icin hazir bir temel olusturma

## Siradaki Gorev

Kullaniciya verilen kucuk alistirma:

1. `index.html` icindeki ana basligi `My City Explorer` yap
2. `Proje Konsepti` kartina kendi cumlenle bir satir ekle
3. `script.js` icine bir adet daha `console.log()` ekle

Bu gorev bittiginde sonraki adima gecilecek:

`Adim 2 - JavaScript temel yapilar (degiskenler, fonksiyonlar, arrayler, donguler)`
